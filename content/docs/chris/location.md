# Location

| Requirement | Pri |
|-------------|:---:|
| 30 min or less commute to Polaris area | ● |
| Near running/biking trails (ride-out access) | ● |
| Near Aldi or Aldi on commute path | ● |
| Not on a busy street | ● |
| Semi-private back yard | ◐ |

*Trails should be bike-friendly without having to load up and drive.*
